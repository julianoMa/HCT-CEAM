Activer le mode maintenance ?

Tout le monde, sauf le président CEAM et l'administrateur, verra une page de maintenance à la place du site — y compris les personnes déjà connectées. Pense à le désactiver une fois ton intervention terminée.

Zone dangereuse

Réinitialise complètement la base de données : tous les dossiers, comptes utilisateurs, notifications, pièces jointes et ce journal d'activité seront supprimés définitivement. Les identifiants (références de dossiers, etc.) repartiront de zéro. Le règlement CEAM est conservé.

Réinitialiser la base de données ?

Cette action est irréversible. Tous les dossiers, comptes, notifications et pièces jointes seront supprimés définitivement, et les compteurs d'identifiants repartiront à zéro.

Confirmation finale

Seront supprimés définitivement :

  • Tous les dossiers CEAM et leurs pièces jointes
  • Tous les comptes utilisateurs (tout le monde devra se reconnecter)
  • Toutes les notifications
  • Le journal d'activité (y compris cette action elle-même, qui ne sera donc pas tracée)

Conservé : le règlement CEAM.

Un code de confirmation à 5 caractères sera ensuite envoyé par MP Discord — la base ne sera réinitialisée qu'après l'avoir retapé.

Dernière étape

Un code à 5 caractères vient de t'être envoyé par MP Discord. Retape-le ici pour réinitialiser définitivement la base de données. Le code expire dans 10 minutes.
